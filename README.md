# Service Budget

Aplicativo mobile desenvolvido com **React Native + Expo** para gerenciamento de **orçamentos de serviços**.
O objetivo do projeto é permitir registrar, visualizar e organizar orçamentos de forma simples e rápida.

## 🚀 Tecnologias

* React Native
* Expo
* JavaScript / TypeScript
* Node.js
* npm ou pnpm

## 📱 Funcionalidades

* Criar novos orçamentos de serviço
* Listar orçamentos cadastrados
* Visualizar detalhes de um orçamento
* Atualizar informações de orçamento
* Remover orçamentos
* Interface mobile simples e responsiva

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/caiofsena/service-budget.git
```

Entre na pasta do projeto:

```bash
cd service-budget
```

Instale as dependências:

```bash
npm install
```

ou

```bash
pnpm install
```

## ▶️ Executando o projeto

Inicie o servidor do Expo:

```bash
npx expo start
```

Depois você pode abrir o aplicativo em:

* 📱 **Expo Go** no celular
* 🤖 **Emulador Android**
* 🍎 **Simulador iOS**

## 📦 Build do aplicativo

Para gerar build usando Expo:

```bash
npx expo prebuild
```

ou usando **EAS Build**:

```bash
npx expo run:android
```

## 🧪 Testes

Caso utilize testes no projeto:

```bash
npm run test
```
